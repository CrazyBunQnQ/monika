成熟 React Dock 组件库对比

适合实现 VS Code 风格动态面板系统

A

dockview

最推荐。零依赖 Docker 布局引擎，支持 Tab、拖拽重组、浮动面板、Popout 独立窗口、JSON 序列化。React/Vue/Angular 多框架。

⭐ ~3,100 | v5.2.0 (2026.3) | MIT | dockview.dev

B

FlexLayout

成熟的 React 停靠布局，支持 Tab、拖拽、浮动面板、序列化。React 专用，久经考验。

⭐ ~1,275 | 2026.3 更新 | MIT

C

rc-dock

轻量级 React Dock 布局，Tab 管理、嵌套拆分、触屏手势。最轻量但功能也最少。

⭐ ~800 | 2025.10 更新 | MIT

关键差异

Dockview 独特优势

  • dockview-core 纯引擎 + React wrapper，架构清晰
  • 支持 Popout（tab 拖出变成独立浏览器窗口）
  • Tab 超出自动溢出菜单（与现有 TabBar 行为匹配）
  • 8 套内置主题，可自定义

需要验证的问题

  • Header 区域能否自定义（放我们的 ModelPicker 等）
  • 能否嵌入现有 Zustand store 而非替换
  • WebView2 兼容性（Wails 桌面环境）
  • 每个 Panel 的内容渲染性能（CodeMirror 实例）
