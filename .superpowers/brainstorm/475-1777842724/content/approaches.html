问题

  • 每个组件重复 resize 逻辑
  • 布局结构硬编码，无法调整
  • 标题栏、TabBar 不统一
  • 添加新面板需要改 App.tsx
A

渐进式 — Panel 抽象 + 固定布局

抽取通用 Panel 组件（TabBar + content + resize 配置），但布局结构在 App.tsx 中预定义。最快落地，但布局不可动态调整。

工期: ~1-2天 | 风险: 低 | 灵活度: 低

B

折中 — SplitTree 容器 + 可配置布局

Panel 抽象 + 树形 Split 容器（水平/垂直分割），初始布局通过 JSON 配置定义，运行时不可拖拽重组但布局随时可改配置。

工期: ~3-5天 | 风险: 中 | 灵活度: 中

C

完整版 — 动态 Dock 系统 (VS Code 风格)

Panel 可拖拽移动/拆分/合并，支持 Tab Group（同区域多面板堆叠），运行时完全动态。最灵活但工程量最大。

工期: ~2-3周 | 风险: 高 | 灵活度: 高
