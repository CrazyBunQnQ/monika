Project Dropdown — 三种布局方案

标题栏左侧，点击项目名弹出下拉菜单

A

简洁列表

类似 VS Code 的 Recent 列表：项目名 + 路径，hover 高亮，底部固定"Open Project..."按钮。最简实现。

B

分组列表

列表按时间分组（Today / Yesterday / This Week / Older），便于查找。适合列表较长时。

C

搜索 + 列表

顶部带搜索输入框，输入过滤列表。列表项右侧有移除按钮（X）。项目极多时最实用。

Branch Dropdown — 同样三种方案

标题栏分支名点击弹出，额外包含创建分支入口

D

简洁列表

分支名列表，当前分支高亮 + checkmark，底部 "New Branch..." + 输入框弹出。

E

分组列表

分支按类型分组（Local Branches / Recent Branches），创建分支入口在每组上方。

F

搜索 + 列表

顶部搜索输入框过滤分支名，新建分支入口在底部。适合分支较多的项目。
