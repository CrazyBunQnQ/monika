数据流与持久化

前端 store → Wails RPC → Go App → provider engine

启动加载流程

1. initProject()   → App.GetCurrentProject()
2. loadProviders()   → App.GetProviders() → ["deepseek", "openai"]
3. loadModels(selectedProvider)   → App.GetModels("deepseek") → [{ID, DisplayName, ContextLimit}, ...]
4. auto-select   若 cfg.model 非空且在当前模型列表中 → 选中它
  否则选中第一个可用模型
5. 用户切换   → setSelectedProvider(p) + loadModels(p) → 自动选中首模型

SendMessage 调用

// 当前签名
App.SendMessage(projectPath, sessionID, text, model)

// 新签名
App.SendMessage(projectPath, sessionID, text, provider, model)

持久化

时机 操作
用户在 picker 中选择模型 App 更新 cfg.ModelProvider + cfg.Model，写入 ~/.monika/config.yaml
每次 SendMessage 调用 同上，确保最后使用的 supplier/model 被记住
下次启动 cfg.ModelProvider 和 cfg.Model 作为默认值，初始化对应的引擎并选中

availableProviders: string[]
selectedProvider: string
modelsByProvider: Record<string, ModelInfo[]>

loadProviders: () => Promise<void>
setSelectedProvider: (id: string) => void

// 修改
loadModels: (providerID: string) => Promise<void>
