交互行为

打开

点击触发按钮 → popover 在按钮上方弹出（placement: top-start），搜索框自动获得焦点。

选择

点击某行 → 同时设置 selectedProvider 和 selectedModel → 关闭 popover → 持久化到 config.yaml。

搜索

输入文字 → 实时过滤（按模型名和供应商名匹配）。无匹配时显示 "No models found"。

关闭

Escape 键 / 点击 popover 外部 / 选择模型后自动关闭。

键盘导航

↑↓ 键移动高亮，Enter 确认选择。

边界情况

无供应商配置：按钮显示灰色 "No providers"，点击无反应。

单供应商：不显示供应商分组头，只列出模型。

某供应商无模型：跳过该供应商，不显示在列表中。
