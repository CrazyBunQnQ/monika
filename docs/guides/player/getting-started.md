# 玩家快速入门

**版本**: v1.0
**最后更新**: 2026-02-07

---

## 欢迎使用 CoC 跑团平台！

这是克苏鲁的呼唤（Call of Cthulhu）7th Edition 的在线跑团平台。本指南将帮助你快速开始游戏。

---

## 第一步：创建账号

1. 访问平台首页
2. 点击"注册"按钮
3. 填写用户信息：
   - 用户名
   - 邮箱
   - 密码
4. 点击"创建账号"

---

## 第二步：创建角色卡

### 方式一：手动创建

1. 登录后，点击"创建角色"
2. 填写角色信息：
   - 姓名
   - 职业
   - 年龄
   - 背景
3. 分配属性（可以随机生成）
4. 选择技能
5. 保存角色卡

### 方式二：导入角色卡

如果你有现成的角色卡：
1. 点击"导入角色"
2. 上传 JSON 文件
3. 确认导入

---

## 第三步：加入游戏

### 寻找游戏

1. 点击"浏览游戏"
2. 查看可用的游戏列表
3. 选择感兴趣的游戏
4. 点击"申请加入"
5. 等待 KP 批准

### 创建游戏

如果你想当 KP：
1. 点击"创建游戏"
2. 上传场景包（或使用示例场景）
3. 设置游戏信息：
   - 游戏名称
   - 最大玩家数
   - 预计时长
4. 生成邀请码分享给朋友

---

## 第四步：开始游戏

### 基础操作

**输入命令**：
- 在输入框中输入命令
- 按回车发送
- 例如：`/roll library_use`

**查看状态**：
```
/status
```
显示你当前的角色状态

**查看可选行动**：
```
/leads
```
显示你当前可以做的行动

---

## 常用命令

### 检定命令

```bash
/roll <技能名>          # 技能检定
/roll 图书馆使用         # 中文技能名
/roll spot_hidden +1    # 带奖励骰
```

**检定结果说明**：
- 🌟 大成功：意外的好运
- ✅ 极难成功：非常成功
- 👍 困难成功：困难地成功
- ✅ 普通成功：成功
- ❌ 失败：失败
- 💀 大失败：意外的灾难

### 推骰和花幸运

```bash
/push          # 推骰（检定失败后可用）
/luck          # 花幸运改变结果
```

### 查询规则

```bash
/rule <问题>    # 查询规则
/rule 暗视      # 查询暗视技能
```

## 界面说明

### 消息区

- 显示游戏剧情和对话
- 你的消息在右侧
- KP 和 NPC 的消息在左侧
- 私密消息带有 🔒 图标

### 状态面板

- 显示你的 HP、SAN、幸运值
- 点击展开查看详细信息
- 移动端：从底部滑出
- 桌面端：固定在右侧

### 骰子结果

- 大号数字显示掷骰结果
- 彩色图标表示成功等级
- 动画效果增强氛围

## 游戏流程

### 1. 探索阶段

```
KP 描述场景 → 你查看 /leads → 选择行动 → /roll 检定
```

### 2. 遭遇战斗

```
战斗开始 → /combat action → 回合循环 → 战斗结束
```

### 3. 总结经验

```
获得奖励 → 可能的成长 → 保存进度
```

## 提示和技巧

### 新手建议

1. **多用 /leads**: 不知道做什么时，先查看可选行动
2. **查询规则**: 不确定规则时，用 `/rule` 查询
3. **勇敢检定**: 不要害怕失败，失败也是故事的一部分

### 角色扮演建议

1. **代入角色**: 用第一人称描述你的行动
2. **参与剧情**: 主动探索和互动
3. **享受故事**: 专注于故事体验，而非"赢"

## 常见问题

### Q: 检定失败了怎么办？

A:
- 失败可以推骰 `/push`（消耗幸运）
- 失败也会推动剧情发展
- 跟 KP 沟通寻找替代方案

### Q: 可以查看其他玩家的信息吗？

A:
- 只能看到公开信息
- 私密信息（如 SAN 值）只有 KP 和你自己可见
- 线索可以设置为私密

### Q: 如何与 KP 沟通？

A:
- 使用 `/msg` 发送私下消息（如果支持）
- 在 Discord/群里沟通（线下游戏）
- 游戏内通过对话输入
