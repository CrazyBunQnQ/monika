## 命令参考格式

```markdown
## /roll - 技能检定

### 描述
进行指定技能或属性的检定，判定成功等级。

### 语法
```
/roll [skill|attribute] [options]
```

### 参数
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| skill | string | 否 | 技能名称或属性名 |
| --difficulty | string | 否 | 难度 (regular/hard/extreme) |
| --modifier | number | 否 | 修正值 |

### 权限
- Player: 可检定自己的角色
- KP: 可检定任何角色

### 示例
```
/roll 侦查
/roll STR
/roll 侦查 --difficulty hard --modifier -10
```

### 响应
成功时返回成功等级和掷骰结果。

### 相关命令
- /push - 推骰
- /luck - 花幸运

### 参见
CoC 7e 规则书 第 58 页
```
