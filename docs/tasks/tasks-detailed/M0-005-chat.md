## 聊天命令规范

### 基本语法

```
/msg <内容>           # 公共聊天
/pm <目标> <内容>     # 私聊
/kp <内容>            # KP 广播
/me <动作>            # 表情动作
/narr <描述>          # 叙述
```

---

## 公共聊天

### /msg

```
用法: /msg <内容>
别名: /m, /say, /s

描述:
  发送公共消息到当前场景，所有玩家可见。

参数:
  内容   消息文本

示例:
  /msg 我觉得这里有问题
  /m 门后好像有声音
```

---

## 私聊

### /pm

```
用法: /pm <目标> <内容>
别名: /whisper, /w, /tell

描述:
  向指定目标发送私密消息，仅双方可见。

参数:
  目标   接收者名称或 ID
  内容   消息文本

示例:
  /pm 张三 我有一个秘密
  /w 守卫A 给我开门
```

---

## KP 广播

### /kp

```
用法: /kp <内容>

描述:
  KP 专用命令，向所有玩家广播 KP 消息。

参数:
  内容   消息文本

示例:
  /kp 所有玩家进行侦查检定
  /kp 场景切换到第二章
```

---

## 表情动作

### /me

```
用法: /me <动作>
别名: /emote, /em

描述:
  描述角色的动作或表情，以第三人称形式显示。

参数:
  动作   动作描述

示例:
  /me 仔细检查桌子
  /me 向守卫点头致意

输出:
  * 张三 仔细检查桌子
```

---

## 叙述

### /narr

```
用法: /narr <描述>
别名: /desc, /n

描述:
  KP 专用，添加场景叙述描述。

参数:
  描述   叙述文本

示例:
  /narr 房间昏暗潮湿，墙角有霉斑
  /desc 守卫看起来很疲惫
```

---

## OOC (Out of Character)

### /ooc

```
用法: /ooc <内容>
别名: //, /brb

描述:
  发送 OOC（角色外）消息，用于玩家间沟通。

参数:
  内容   消息文本

示例:
  /ooc 我需要离开一下
  // 那个规则是怎么说的？
```

## 输出格式

### 公共消息

```
[张三]: 我觉得这里有问题
```

### 私聊

```
[私聊] 你 -> 张三: 我有一个秘密
```

### 表情动作

```
* 张三 仔细检查桌子
```

### KP 广播

```
[KP]: 所有玩家进行侦查检定
```

### OOC 消息

```
((OOC)) 李四: 我需要离开一下
```
