## 角色命令规范

### 基本语法

```
/character [子命令] [参数]
/char [子命令] [参数]
/switch <角色名>
/set <属性> <值>
/skill <技能名> [动作] [值]
```

---

## 角色查看

### /character list

```
用法: /character list
别名: /char list, /chars

描述:
  显示当前用户的角色列表。

示例:
  /character list
  /char list
```

输出示例：
```
你的角色：
1. 张三 - 私家侦探 (HP: 11/11, SAN: 60/60)
2. 李四 - 医生 (HP: 10/11, SAN: 55/60)

使用 /switch <角色名> 切换当前角色
```

### /character view

```
用法: /character view [角色名]
别名: /char view

描述:
  查看指定角色的详细信息。

参数:
  角色名   可选，默认为当前角色

示例:
  /character view          # 查看当前角色
  /character view 张三     # 查看指定角色
```

---

## 角色切换

### /switch

```
用法: /switch <角色名>
别名: /use, /activate

描述:
  切换当前使用的角色。

参数:
  角色名   要切换到的角色名称

示例:
  /switch 张三
  /use 李四
```

---

## 属性设置

### /set hp

```
用法: /set hp <角色名> <值>
别名: /hp

描述:
  设置角色的生命值。

参数:
  角色名   目标角色，默认为当前角色
  值       HP 值，可以是绝对值或相对值

示例:
  /set hp 10              # 设置当前角色 HP 为 10
  /set hp 张三 -1         # 当前角色 HP -1
  /set hp 李四 +1d4       # 掷骰恢复 HP
```

### /set mp

```
用法: /set mp <角色名> <值>

描述:
  设置角色的魔法值。

示例:
  /set mp 8               # 设置 MP 为 8
  /set mp -2              # 消耗 2 点 MP
```

### /set san

```
用法: /set san <角色名> <值>

描述:
  设置角色的理智值。

示例:
  /set san 55             # 设置 SAN 为 55
  /set san -5             # SAN -5
```

### /set attribute

```
用法: /set attribute <角色名> <属性> <值>

描述:
  设置核心属性（临时或永久）。

参数:
  属性     STR/CON/SIZ/DEX/APP/INT/POW/EDU 之一

示例:
  /set attribute 张三 DEX 60    # 设置敏捷
  /set attr 李四 POW -2         # 意志 -2（临时）
```

## 技能管理

### /skill check

```
用法: /skill check <技能名>
别名: /sk check

描述:
  查看技能的详细信息。

示例:
  /skill check 侦查
```

### /skill add

```
用法: /skill add <角色名> <技能名> <值>

描述:
  添加新技能到角色卡。

示例:
  /skill add 张三 驾驶 30
```

### /skill improve

```
用法: /skill improve <角色名> <技能名> [增加值]

描述:
  提升技能值。增加值默认为 1d10-1d6。

示例:
  /skill improve 张三 侦查      # 正常提升
  /skill improve 张三 侦查 +2    # 固定 +2
```

### /skill roll

```
用法: /skill roll <角色名> <技能名>

描述:
  使用指定技能进行检定。

示例:
  /skill roll 张三 侦查
```
