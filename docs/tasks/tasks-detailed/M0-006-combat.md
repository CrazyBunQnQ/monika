## 战斗命令规范

### 基本语法

```
/attack [目标] [伤害]
/damage <目标> <伤害> [类型]
/heal [目标] <恢复量>
```

---

## 攻击命令

### /attack

```
用法: /attack [目标] [伤害骰]
别名: /atk, /hit

描述:
  执行攻击检定，可选择目标和伤害骰。

参数:
  目标      可选，攻击的目标名称
  伤害骰    可选，武器伤害如 1d6+2

示例:
  /attack                   # 仅进行攻击检定
  /attack 怪物A             # 攻击指定目标
  /attack 怪物A 1d6+2       # 攻击并计算伤害
  /atk 守卫 1d8+3           # 使用别名
```

### 攻击流程

```
1. 进行攻击检定（斗殴或射击技能）
2. 如果成功，掷伤害骰
3. 扣除目标 HP
4. 显示战斗结果
```

## 伤害命令

### /damage

```
用法: /damage <目标> <伤害值> [伤害类型]
别名: /dmg, /hurt

描述:
  直接对目标造成伤害，不进行检定。

参数:
  目标      伤害目标（必需）
  伤害值    伤害骰或固定值（必需）
  伤害类型  可选，如穿刺、钝击、火炎等

伤害类型:
  穿刺     剑、匕首等
  钝击     棍棒、拳头等
  斩击     斧、大刀等
  火炎     火焰伤害
  电击     电流伤害
  冷气     冰冻伤害
  毒素     中毒伤害

示例:
  /damage 怪物A 1d6+2
  /damage 守卫 8 穿刺
  /dmg 怪物B 2d8 火炎
```

## 治疗命令

### /heal

```
用法: /heal [目标] <恢复量>
别名: /cure, /recover

描述:
  恢复目标的生命值。

参数:
  目标      可选，默认为当前角色
  恢复量    恢复的 HP 数量（必需）

示例:
  /heal 5                 # 恢复自己 5 点 HP
  /heal 队友B 10          # 恢复队友B 10 点 HP
  /cure 张三 1d6+2        # 掷骰恢复
```

## 战斗状态

### 战斗状态跟踪

```
状态名称 | 效果
---------|------
昏迷     | HP ≤ 0，无法行动
死亡     | HP < -最大值
恐惧     | 检定修正 -1d4
狂暴     | 伤害 +2，防御 -2
中毒     | 每回合 -1 HP
麻痹     | 无法行动
```

## 伤害减免

### DR (Damage Reduction)

```
/attack 怪物A 1d6+2 DR2

说明:
  DR2 = 伤害减免 2
  最终伤害 = 掷出伤害 - 2
```
