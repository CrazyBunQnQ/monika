### 基本语法

```
/inventory            # 查看背包
/item <操作> <物品>   # 物品操作
/equip <装备>         # 装备物品
/unequip <装备>       # 卸下装备
```

## 物品查看

### /inventory

```
用法: /inventory [类别]
别名: /inv, /bag, /i

描述:
  显示角色的物品清单。

参数:
  类别   可选，过滤物品类型

示例:
  /inventory              # 显示所有物品
  /inv 武器               # 只显示武器
  /i 消耗品               # 只显示消耗品
```

### 物品类别

```
武器      - 近战/远程武器
防具      - 护甲、盾牌
消耗品    - 药品、弹药
工具      - 工具包、装备
道具      - 特殊物品
杂物      - 其他物品
```

## 物品操作

### /item

```
用法: /item <子命令> [参数]
别名: /it

子命令:
  add     添加物品
  remove  删除物品
  use     使用物品
  give    给予物品
  drop    丢弃物品
```

#### /item add

```
用法: /item add <物品名> [数量]

描述:
  添加物品到背包。

示例:
  /item add 手电筒
  /item add 子弹 12
  /it add 急救包 2
```

#### /item remove

```
用法: /item remove <物品名> [数量]

描述:
  从背包移除物品。

示例:
  /item remove 手电筒
  /item remove 子弹 5
```

#### /item use

```
用法: /item use <物品名>

描述:
  使用消耗品或道具。

示例:
  /item use 急救包
  /item use 手电筒
```

#### /item give

```
用法: /item give <目标> <物品名> [数量]

描述:
  给予物品给其他角色。

示例:
  /item give 张三 子弹 6
  /it give 队友A 急救包
```

#### /item drop

```
用法: /item drop <物品名> [数量]

描述:
  丢弃物品到当前场景。

示例:
  /item drop 空弹夹
```

## 装备系统

### /equip

```
用法: /equip <装备名>
别名: /eq

描述:
  装备武器或防具。

装备槽位:
  主手     - 单手武器
  副手     - 副手武器或盾牌
  双手     - 双手武器
  身体     - 护甲
  头部     - 头盔
  饰品     - 饰品

示例:
  /equip 左轮手枪
  /eq 防弹背心
```

### /unequip

```
用法: /unequip <槽位>
别名: /ueq

描述:
  卸下装备。

示例:
  /unequip 主手
  /ueq 身体
```
