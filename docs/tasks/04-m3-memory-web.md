#### 恢复流程

```
1. 暂停 Session
   - 保存完整状态快照
   - 标记 Session 为 paused
   - 清理 WebSocket 连接

2. 恢复 Session
   - 验证用户身份
   - 加载最近检查点
   - 同步到最新事件
   - 重建 WebSocket 连接

3. 冲突处理
   - 如果有后续事件：
     - 提示用户选择
     - 展示差异
     - 合并或覆盖
```
