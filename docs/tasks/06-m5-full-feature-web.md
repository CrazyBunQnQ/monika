#### SAN 规则

```

1. KP 描述恐怖场景
2. 系统查找对应 SAN 损耗
3. 玩家进行 SAN 检定 (POW*5)
4. 根据结果扣除 SAN:
   - 成功: 扣除 0-1 点
   - 失败: 扣除 1-5 点
   - 大失败: 扣除最大值
```
