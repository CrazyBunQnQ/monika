#### 检定规则

```
成功等级:
  - 大成功 (CR): 掷骰 <= 1/5 成功值
  - 极难成功 (EXT): 掷骰 <= 1/5 成功值
  - 困难成功 (HRD): 掷骰 <= 1/2 成功值
  - 普通成功 (REG): 掷骰 <= 成功值
  - 失败 (FAIL): 掷骰 > 成功值
```
